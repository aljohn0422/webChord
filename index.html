Chord Harmonica Web Simulator

線上和弦口琴模擬器以 Web Audio API 開發。所有聲音皆由 oscillator 發聲，故聽起來是很不真實的「電子」的音色。和弦內音的排列採Suzuki和弦口琴配置，是轉位過的和弦，完整的音符列表請上網查詢Suzuki和弦口琴，或是進入Github看看我亂七八糟的程式碼。

和弦口琴中每個和弦完整的八個音皆有發聲，其中，高頻有稍微濾波，使音色較圓潤。但透過 oscillator 來發聲實在不是什麼好方法，音色聽起來與實際的聲音相差甚遠，下一步要研究 Web MIDI API，使網頁可以發出鋼琴或其他較接近真實樂器的聲音。

This tool works best with Google Chrome.
